java命令启动

java命令启动

逼逼了这么多终于可以开始了是吗？

嗯嗯是的呢，但是你当着我的面说这种话的话就直接拉黑了嗷

启动命令

最简单的启动命令就是java -jar

启动服务器

现在，以purpur1.18.2服务端为例，相信你已经下载好了核心，一个后缀为jar的文件

哈？你不知道怎么下载？关服吧

现在你把下载好的服务端核心放到一个空文件夹里

单击这个位置，输入cmd然后回车，他会蹦出来一个黑框子

如果你操作无误的话，他会显示出你的路径， > 后面是让你输入命令

命令：

java -jar 核心名

如果说，你看到你的核心没有 .jar 这个后缀，点这里

回车执行命令后，你的服务器应该启动了

[时间 ERROR]: Failed to load properties from file: server.properties

[时间 WARN]: Failed to load eula.txt

[时间 INFO]: You need to agree to the EULA in order to run the server. Go to eula.txt for more info.

这啥意思？

打开这个目录下的eula.txt

eula=false 改成 eula=true 然后保存

接着重新点击bat 启动你的服务器

当你看到 Done (9.254s)! For help, type "help"

这玩意，意思就是说你服务器成功开启了

eula是个什么

就是用户协议那种东西

点这里看Eula | Minecraft
